# Customer Churn Analysis (Python)

**Clean, beginner-friendly customer churn analysis with clear EDA and insights.**

---

## Overview
This repository contains a concise Customer Churn Analysis implemented in Python. The project focuses on data cleaning, step-by-step exploratory data analysis, and clear visualizations to explain why customers churn — with no complex machine learning models required.

---

## Contents
- `customer_churn_analysis.ipynb` — Main Jupyter notebook with full analysis (step-by-step).
- `customer_churn_report.pdf` — Polished PDF summarizing findings and business recommendations.
- `data/customer_churn.csv` — Dataset used (included only if licensing allows).
- `charts/` — Key plot images used in the report (optional).
